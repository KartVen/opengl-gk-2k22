# OpenGL 2k22 Project using GLUT library

## How to try it?

- Clone the repository:
```bash
git clone
```

- Run project's solution file:
```
opengl_gk_2k22.sln
```

- Compile and Run project

## Minimal requirements
- [Visual Studio 2019](https://visualstudio.microsoft.com/pl/downloads/)

## References
